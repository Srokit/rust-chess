# Instructions

Install [Rust And Cargo](https://www.rust-lang.org/learn):
```
$ curl https://sh.rustup.rs -sSf | sh
```

Clone this repo.

From terminal:

```
$ cd <repo_root_dir>
$ cargo run
```
